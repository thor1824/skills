## Agent skills

### Issue tracker

Issues for this repo are tracked as local markdown files under `.scratch/<feature-slug>/`. See `docs/agents/issue-tracker.md`.

### Triage statuses

Issue workflow state is recorded in front matter `status` using the repo's standard seven-status vocabulary. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain-doc layout with a root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
